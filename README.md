# MusicPlayer

#### 介绍
调用酷狗api的音乐播放器，实现功能如下：
1.导入本地音乐播放
2.播放、暂停控制
3.音乐进度条
4.音量控制
5.网络音乐搜索、播放
6.下载网络音乐到本地

注：由于api问题有些网络音乐媒体不完整
